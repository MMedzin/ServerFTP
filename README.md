# ServerFTP
## Maria Mikołajczak
## Michał Mędzin

### Description
Simple ftp server based on BSD sockets. Server is compliatn with ```RFC 959``` standard.

#### Supported ftp commands
```USER```, ```PASS```, ```SYST```, ```PWD```, ```TYPE``` (ASCII and IMAGE), ```PORT```, ```LIST```, ```QUIT```, ```RMD```, ```CWD```, ```CDUP```, ```MKD```, ```STOR```, ```DELE```, ```RETR```
